[ENG] How to change the header logo picture:
--------------------------------------------
1. Open the file named `/src/config.ts` in your code editor.
2. Find the line:
   `HEADER_LOGO_URL: "https://nirob-free-fire-baner.vercel.app/profile?uid=2279016714",`
3. Replace the URL inside the quotes with your own picture URL (e.g. Imgur link, Discord image link, or Garena profile URL).
4. Save the file. The header logo next to "AYAN'S FF VIP HUB" will update instantly!
